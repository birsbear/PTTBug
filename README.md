# PTTBug
PTT爬蟲小程式
簡單的爬PTT想要看的板有新增那些文

功能:

    1.指定想要查看的板
  
    2.顯示選擇的板想要幾頁的內容

  
預計新增功能:

    1.看指定板有沒有新文章(以上次來此版為基準)
  
    2.定時爬取文章
  
    3.有新文章即時顯示文章標題和提醒
